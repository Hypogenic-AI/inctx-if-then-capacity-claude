# Downloaded Papers

Papers relevant to the "In-context If-Then Capacity" research project.

## Core Papers (Instruction Following Evaluation)

1. **IFEval: Instruction-Following Evaluation for Large Language Models** (zhou2023_ifeval.pdf)
   - Authors: Zhou, Lu, Mishra, Brahma, Basu, Luan, Zhou, Hou
   - Year: 2023 | arXiv: 2311.07911
   - Why relevant: Foundational benchmark for verifiable instruction following; 25 constraint types

2. **The SIFo Benchmark: Sequential Instruction Following** (chen2024_sifo.pdf)
   - Authors: Chen, Liao, Qi, Eustratiadis, Monz, Bisazza, de Rijke
   - Year: 2024 | arXiv: 2406.19999
   - Why relevant: Tests sequential conditional instruction following; security rules task is directly conditional

3. **FollowBench** (jiang2023_followbench.pdf)
   - Authors: Jiang et al.
   - Year: 2023 | arXiv: 2310.07641
   - Why relevant: Multi-level constraint following with escalating difficulty

4. **COLLIE: Constrained Text Generation** (yao2023_collie.pdf)
   - Authors: Yao, Chen, Hanjie, Yang, Narasimhan
   - Year: 2023 | arXiv: 2307.06439
   - Why relevant: Grammar-based framework for compositional constraints

5. **InFoBench** (qin2023_infobench.pdf)
   - Authors: Qin et al.
   - Year: 2023 | arXiv: 2311.07397
   - Why relevant: Decomposed instruction evaluation methodology
